# Text Classification using NLP: Question vs Statement Detection

## Objective
To classify text into questions and statements using NLP and machine learning techniques.

## Tools Used
- Python (NLTK, Scikit-learn)
- TF-IDF Vectorization
- Machine Learning Models

## Methodology
- Text cleaning and preprocessing
- Tokenization and lemmatization
- TF-IDF feature extraction
- Model training with Logistic Regression, KNN, Decision Tree

## Model Performance
(Logistic Regression performed best with ~69% accuracy)

## Key Insights
- Logistic Regression provided best balanced performance
- High-dimensional sparse data affects tree-based models

## Business Impact
- Improves chatbot understanding
- Enables automation in customer support
- Helps classify user intent

## Conclusion
Traditional ML models can effectively classify text, but advanced NLP techniques can further improve performance.
